# Island Connect AI Blog - Jamaican Brand Colors & AI Revolution

## 🏝️ Project Overview

This is a comprehensive blog website for **Island Connect AI** that showcases Jamaican brand colors (green, gold, black) and explores how AI agents and smart websites are transforming the real estate and tourism industries. The website features a dedicated blog section with multiple topics, interactive elements, and data visualizations.

- **Chart.js** for data visualizations
- **Vanilla JavaScript** for interactivity
- **Font Awesome** for icons
- **Google Fonts** (Inter family)

